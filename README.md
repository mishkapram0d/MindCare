# MindCare – Student Mental Health Support

MindCare is a web application designed to support students' mental well-being by providing mood tracking, mental health resources, AI-powered support, and emergency assistance.

## Features

- Mood Tracking
- AI Support Chat
- Mental Health Resources
- Emergency Support
- Responsive User Interface

## Technologies Used

- React
- Vite
- JavaScript
- HTML5
- CSS3
- Netlify
